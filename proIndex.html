HTML

  • Hyper Text Markup Language
  • Standar Markup Language for Creating Website/web page
  • Describes the structure of a web page
  • Consist of series of Element
  • Element tells the browser how to display the content

HTML ELEMENTS :

  • Element is a components or node, it is used to represent different type of content.
  • Elements is defined by "Start Tag" and "Main Content to Display" and "End Tag".
  • Element's content is in between "Start tag" name and "End tag name"
